# 📊 CodeAlpha - Netflix Data Visualization

## Project Overview
This project was completed as part of the CodeAlpha Data Analytics Internship.

The objective of this project is to visualize the Netflix Movies and TV Shows dataset using Python and Matplotlib.

## Features
- Movies vs TV Shows Bar Chart
- Top 10 Countries Bar Chart
- Content Ratings Pie Chart
- Release Year Histogram
- Top 10 Genres Bar Chart

## Technologies Used
- Python
- Pandas
- Matplotlib

## Dataset
Netflix Movies and TV Shows Dataset

## Output
The generated charts are saved in the `output` folder.
